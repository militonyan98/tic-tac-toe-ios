# TIC-TAC-TOE - iOS SwiftUI


## Main functionality

* Classic Tic-Tac-Toe game
* Single-player - play against AI
* Three difficulty levels: easy, medium, and hard
* Two-player


### *Single-player*

* Uses Minmax algorithm
* Difficulty level implemented by depth pruning


## Technologies Used

* SwiftUI
